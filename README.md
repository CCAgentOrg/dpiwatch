# 📡 DPI Watch

An AI-powered, fully automated daily newsletter tracking India's Digital Public Infrastructure (DPI) ecosystem. Content is generated by autonomous agents and published to GitHub Pages.

## 🌐 Live Site

**URL**: https://digital-india.cashlessconsumer.in

## 🤖 How It Works

This blog is maintained by **2 autonomous AI agents** running on a schedule:

| Agent | Schedule | Coverage | Output |
|-------|----------|----------|--------|
| **Daily DPI Brief** | Daily @ 8:00 AM IST | Last 24 hours | All 7 DPI layers |
| **Themed Deep Dive** | Daily @ 8:30 AM IST | Last 7 days | Rotating themes |

### DPI Layers Covered

1. **Identity & Auth** — Aadhaar, PAN, DigiLocker, eSign, eKYC
2. **Payments & Financial Rails** — UPI, BBPS, NFS, RTGS/NEFT, TReDS
3. **Data Exchange** — API Setu, Health Stack, Agri Stack
4. **Commerce & Logistics** — ONDC, ULIP, eWAY, FASTag
5. **Sectoral Infrastructure** — ABHA, NeVA, eCourt, AISHE
6. **Governance & G2C** — eOffice, CPGRAMS, eDistrict, UMANG
7. **Security & Trust** — CCA certificates, DPDP Act, CERT-In

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/CCAgentOrg/dpiwatch.git

# Install Hugo
# https://gohugo.io/installation/

# Run locally
hugo server -D

# Build
hugo --minify
```
