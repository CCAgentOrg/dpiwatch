# 📡 DPI Watch

India's Digital Public Infrastructure tracker — monitoring Aadhaar, UPI, ONDC, DigiLocker, and government digital initiatives from a citizen rights perspective.

## 🤖 AI Agents (6 Active)

This blog is fully powered by autonomous AI agents that publish daily:

| Agent | Schedule | Focus | Output |
|-------|----------|-------|--------|
| **Daily DPI Brief** | Daily @ 8:00 AM IST | All 7 DPI layers from last 24 hours | `posts/YYYY-MM-DD-dpi-brief.md` |
| **Themed Deep Dive** | Daily @ 8:30 AM IST | Rotating themes: Mon: Identity, Tue: Payments, Wed: Data Exchange, Thu: Commerce, Fri: Sectoral, Sat: Governance, Sun: Security/Privacy | `posts/YYYY-MM-DD-dpi-deep-dive-[theme].md` |
| **Weekly Deep Research** | Sundays @ 9:00 AM IST | 2000+ word comprehensive analysis on DPI developments | `posts/YYYY-MM-DD-deep-research-[topic].md` |
| **Weekly Satire** | Saturdays @ 10:00 AM IST | Futuristic satire highlighting citizen rights threats | `posts/YYYY-MM-DD-satire-[topic].md` |
| **101 Explainers** | Thursdays @ 9:00 AM IST | Encyclopedic DPI guides with citizen rights analysis | `explainers/101-[topic].md` |

### The 7 DPI Layers
1. **Identity** — UIDAI, Aadhaar, eKYC
2. **Payments** — UPI, NPCI, RuPay
3. **Data Exchange** — DigiLocker, API Setu, eSign
4. **Commerce** — ONDC, network platforms
5. **Sectoral** — Health (ABHA), Agriculture (AgriStack)
6. **Governance** — DARPG, CPGRAMS, e-Office
7. **Trust** — CERT-In, DPDP Act, Data Protection
